# PerfectPlayer
使用框架
为了便捷开发，使用了MVVMLIN框架，感谢
用exoplayer实现的播放本地视频播放器
架构简单，功能简单
MVVM+kotlin+exoplayer实现
当前实现点：
1. 扫描本地视频并以文件夹（视频列表）形式显示
2、针对视频功能：单视频循环，目录循环播放，支持截图与变速，支持硬解，支持快进，快退支持全屏与填充两种方式
